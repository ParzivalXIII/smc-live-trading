Purpose
-------

DataManager sits between the CCXT fetcher/processor and CSV storage.
It manages incremental OHLCV data fetching: determine file path →
load existing → find last timestamp → fetch only missing candles →
merge → dedup → save atomically.


Pipeline Position
-----------------

  CCXT Exchange
       ↓
  fetch_ohlcv()          (trade_scripts/fetcher.py)
       ↓
  process_candles()      (trade_scripts/processor.py)
       ↓
  DataManager.update()   (trade_scripts/data_manager.py)  ← NEW
       ↓
  save_candles()         (trade_scripts/storage.py)
       ↓
  analyze_ta.py → Orchestrator


Architecture
------------

  DataManager is pure composition — it does NOT reimplement any
  module logic. It delegates to:

    ExchangeFactory.create()        → exchange instance
    ExchangeFactory.resolve_symbol() → unified symbol
    fetch_ohlcv()                   → raw candle list
    process_candles()               → normalized DataFrame
    load_candles()                  → existing CSV data
    save_candles()                  → atomic CSV write

  Zero logic reimplementation. Clean separation of concerns.


DataManager.update() Flow
--------------------------

  1. Resolve symbol (e.g. "BTCUSDT" → "BTC/USDT")
  2. Build CSV path: data/ohlcv_{resolved}_{tf}.csv
  3. Load existing CSV (if exists, handle corruption)
  4. Determine fetch_since:
     - If existing data: last_timestamp + 1ms (incremental)
     - If explicit since: use it
     - Otherwise: None (exchange default)
  5. Fetch only missing candles from exchange
  6. Process new candles (sort, dedup, clean)
  7. Merge new + existing (concat → sort → dedup → reset_index)
  8. Save atomically (tempfile + rename)
  9. Return merged DataFrame

Usage
------

  from trade_scripts.data_manager import DataManager

  dm = DataManager()                         # default: bybit, data/

  # First fetch (no CSV exists)
  df = dm.update("BTCUSDT", "4h")
  # → fetches from exchange, saves to data/ohlcv_BTCUSDT_4h.csv
  # → returns DataFrame with all candles

  # Incremental fetch (CSV exists)
  df = dm.update("BTCUSDT", "4h")
  # → reads last timestamp from CSV
  # → fetches only newer candles
  # → merges, deduplicates, saves atomically

  # Backfill with explicit since
  df = dm.update("ETHUSDT", "1d", since=1700000000000)

  # Custom exchange and config
  dm = DataManager(exchange_id="binance", data_dir="custom_data",
                   config={"apiKey": "...", "secret": "..."})
