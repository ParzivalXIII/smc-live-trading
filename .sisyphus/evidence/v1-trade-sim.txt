Architecture
------------

Three-Phase BacktestHarness.run():

  Phase 1 (Streaming)
    _SwingEngine.update(i, row) bar-by-bar
        ↓ swings_df
  Phase 2 (Batch)
    bos_choch(), ob(), liquidity(), retracements()
        ↓ report (26 columns)
  Phase 3 (Strategy Simulation) — NEW
    for each row in report:
        strategy.update(i, row, engine_result, simulator)
            ↓
        TradeSimulator.enter_long() / .close()
            ↓
        result.trades, result.equity_curve, result.metrics

Trade Metrics (9)
------------------

  total_trades     int      Total number of closed trades
  wins             int      Number of winning trades (pnl > 0)
  losses           int      Number of losing trades (pnl < 0)
  win_rate         float    wins / total_trades (NaN if zero trades)
  gross_profit     float    Sum of all positive PnLs
  gross_loss       float    Sum of all negative PnLs (absolute)
  profit_factor    float    gross_profit / gross_loss (1e9 if no losses; NaN if zero trades)
  net_pnl          float    gross_profit - gross_loss
  max_drawdown     float    Peak-to-trough of equity curve (NaN if zero trades)

Key Design Decisions
--------------------

  Decision                    Choice
  ──────────                  ─────
  Strategy execution phase    Phase 3 (after batch) — BOS only available post-batch
  Row content                 Full report row (26 cols: BOS, OB, swings, etc.)
  Entry/exit price            Close of signal bar (V1 simplicity)
  Equity curve                Realized PnL only (no mark-to-market)
  Position model              Single position (0 or 1 — no hedging)
  Simulator in strategy       Passed as argument, not global
  smc.py changes              ZERO (guardrail enforced)


Usage
-----

Python API:

  from backtest import BacktestConfig, BacktestHarness
  from strategies.bos_flip import BOSFlipStrategy

  cfg = BacktestConfig(swing_length=5, confirmation_bars=2)
  strategy = BOSFlipStrategy()
  harness = BacktestHarness(cfg, strategy_callback=strategy)
  result = harness.run("BTCUSDT_4H.csv")

  result.trades         → DataFrame: side, entry/exit, pnl
  result.equity_curve   → Series: cumulative PnL
  result.metrics        → dict: 18 base + 9 trade metrics

Custom strategy:

  class MyStrategy:
      def update(self, index, row, engine_result, simulator=None):
          if row.get("HighLow") == 1:  # swing high
              simulator.enter_short(index, row.name, row["Close"])
